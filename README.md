# Church Website

This project is a one-month church website and digital admin system.

## Project Goals

- Build a public church website for visitors and members.
- Create a private admin area for church family records.
- Track donations safely and clearly.
- Prepare the project for GitHub, Supabase, and Vercel.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase PostgreSQL, planned
- Vercel, planned

## Planned Pages

Public website:
- Home
- About
- Services
- Events
- Sermons
- Contact
- Donate

Private admin system:
- Admin Login
- Dashboard
- Families
- Members
- Donations
- Reports
- Events Admin
